# Meta Ads クリエイティブコピー下書き

> **対象**: Facebook / Instagram Feed 広告（静止画・動画）
>
> **配信目的**: コンバージョン（Lead）
>
> **方針**: 訴求軸ごとに 3 パターン書く。Meta は **クリエイティブ訴求力 = 結果の 90%** なので、文言×画像の組み合わせ多様化が最重要。

---

## クリエイティブ構成要素（Meta フォーマット）

| 要素 | 文字数制限（推奨） | 役割 |
|---|---|---|
| **プライマリテキスト** | 125 文字以内（モバイルで全文表示） | 本文・最重要 |
| **見出し（Headline）** | 27 文字以内 | 強い訴求点 |
| **説明（Description）** | 27 文字以内 | サブ訴求・行動喚起 |
| **CTA ボタン** | 既定文言から選択 | 「詳しくはこちら」「お問い合わせ」「サインアップ」等 |

> **初心者ポイント**: モバイルでは「もっと見る」ボタンの前に **約100文字** しか表示されない。**最初の100文字に最重要メッセージを置く**。

## 画像素材の方針

> **既存 IMAGES_MANIFEST.md の活用**:
> - `feature-support.jpg`（ワンストップサポート）→ パターンA向け
> - `feature-inbound.jpg`（インバウンド対応）→ パターンB（手数料訴求と相性）
> - `feature-cleaning.jpg`（清掃管理）→ パターンA（信頼訴求）
> - `svc-pricing.jpg`（ダイナミックプライシング）→ パターンA（数字訴求）

### 推奨フォーマット

| フォーマット | サイズ | 用途 |
|---|---|---|
| **Feed 正方形** | 1080×1080 | Facebook/Instagram Feed メイン |
| **Feed 縦長** | 1080×1350 | Instagram Feed（より大きく表示） |
| **Stories/Reels** | 1080×1920 | Instagram Stories/Reels |
| **動画 Feed** | 1080×1080, 15-30秒 | Reels と兼用可 |

### テンイチ本人画像の活用提案

> 信頼主導 LP のためのテンイチ本人画像があれば、以下のフォーマットで活用効果大:
>
> - **「代表が答えます」型**: テンイチの顔写真 + 「物件運用のご質問、まずは私が直接お答えします」
> - **動画 15秒**: 一人称で「なぜ手数料8%にできるのか」を語るショート動画
>
> Meta の Reels/Stories は人物動画が **CTR 1.5-2倍** になる傾向。本格的な撮影でなくても **iPhone での自撮り動画**で十分機能する。

## A/B テスト設計

> **重要**: Meta の AI に学習させるため、初月は **広告セット内で複数クリエイティブを並走** させる。Meta が自動で勝者にトラフィックを寄せる。

| 広告セット | 推奨クリエイティブ数 | 入れる組み合わせ |
|---|---|---|
| Retargeting | 3-4 | C-1 + C-2 + C-3（リターゲ向け） |
| Lookalike | 3-4 | A-1 + A-2 + B-1 + B-2 |
| Interest（Airbnb ホスト） | 4-6 | 全パターン1本ずつ |
| Interest（不動産投資家） | 3-4 | A-1 + A-2 + B-1（信頼+価格） |

> **初心者ポイント**: 広告1本に予算を集中するより、3-5本を Meta に渡してダイナミック広告で勝たせるのが王道。

## ランディング URL

```
Feed / Stories / Reels:
  https://sekaistay.com/switch?utm_source=meta&utm_medium=cpc&utm_campaign={campaign_name}&utm_content={ad_name}        ← Control
  https://sekaistay.com/switch-lite?utm_source=meta&utm_medium=cpc&utm_campaign={campaign_name}&utm_content={ad_name}    ← B (lite form)
  https://sekaistay.com/switch/short?utm_source=meta&utm_medium=cpc&utm_campaign={campaign_name}&utm_content={ad_name}   ← D (45min meeting)
  https://sekaistay.com/switch/founder?utm_source=meta&utm_medium=cpc&utm_campaign={campaign_name}&utm_content={ad_name} ← E (founder)
```

LP A/B/C/D/E 振り分けは広告URL でコントロール（広告セット内で variant ごとに URL を分ける）。`lp_variant` は `<PageViewTracker lpVariant="..." />` 経由で GA4 / Meta Pixel / Supabase に伝播済み。

> **初月の Meta 推奨**: 全 5 variants を回すと統計的に弱くなるので、**Control + B + E（founder）の 3 変種**から開始。Founder variant は信頼訴求と相性が良い Meta オーディエンスで効きそう。

## 確認したい項目

1. **テンイチ本人画像/動画の用意**: 後追い実装か、Day 2 launch 時点で間に合うか？
2. **「業界相場の半額」の数字根拠**: パターン B 全体で使うので根拠資料の URL or PDF を確保
3. **「稼働率+18%」のパターン A-1**: この数字の根拠は？ 実績データなら使う、推測ならコピー差し替え必要
4. **広告コピーの最終承認フロー**: テンイチ単独 OK か、誰かのレビュー必要か？
