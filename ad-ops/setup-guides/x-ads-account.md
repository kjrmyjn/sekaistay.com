# X Ads アカウント開設・コンバージョン設定 手順

> **目的**: 開設未完了の X Ads アカウントを Day 3-5 launch 可能な状態にする。
>
> **重要**: 審査に **24-48時間**かかる場合があるため、**他媒体より先に申請**する。
>
> **所要時間**: 申請 20-30 分 + 審査待ち 24-48h + コンバージョン設定 20分

---

## 全体像

```
Step 1: X Ads アカウント申請（即日着手・最重要）
Step 2: 審査待ちの間にやること
Step 3: 審査通過後の Universal Website Tag（X Pixel）設定
Step 4: コンバージョン イベント定義
Step 5: 課金方法の設定
Step 6: 初回キャンペーン Live 化
```

## Step 3: Universal Website Tag（X Pixel）設定

> 審査通過後に実施。Meta Pixel と同じくクライアント側 JavaScript タグ。

### 操作手順

1. X Ads Manager → **「ツール」 → 「コンバージョントラッキング」**
2. **「+ ウェブサイトタグを作成」**
3. タグ名: `Sekai Stay - sekaistay.com`
4. **「Universal Website Tag」**を選択（推奨・Pixel 1個で全イベント計測）
5. 生成されたタグの **「ピクセル ID」**（例: `o1234`）と **タグ コード全体**をコピー
6. **「ID とコード」を Discord で私に共有** → 私が `app/layout.tsx` に追加（PR3 拡張 or PR5 で）

### タグ実装パターン

私の実装方針:
- 既存の `META_PIXEL_ID` と同様のパターンで `X_PIXEL_ID` を環境変数化
- `app/layout.tsx` の `<head>` に追加
- `AnalyticsRouteTracker.tsx` で SPA ナビゲーションごとに PageView 発火

## Step 4: コンバージョン イベント定義

### 操作手順

1. X Ads Manager → **「ツール」 → 「コンバージョントラッキング」 → 「イベント」**
2. **「+ コンバージョンを追加」**
3. 設定:
   - イベント名: **`Lead`**
   - イベントタイプ: **「サイト訪問者の獲得（Lead generation）」**
   - イベント発火条件:
     - **方式 A（推奨・URL ベース）**: フォーム送信後の遷移先 URL `https://sekaistay.com/contact/thanks`（→ サンクスページ作成必要）
     - **方式 B（イベント ベース）**: JavaScript で `twq('event', 'tw-XXX-YYY', {...})` を発火
   - コンバージョンウィンドウ: **30日**
   - 価値: **¥5,000**（Google Ads と統一）
4. **「保存」**

> **初心者ポイント**: 方式 A（URL ベース）はサンクスページが必須。方式 B はコード変更必要。**現状 sekaistay.com にサンクスページがない場合は方式 B が早い**（Web3Forms 送信成功時に発火するコールバックでイベント発火）。
>
> サンクスページは私が PR5 で実装可能。テンイチの方針確認要。

## Step 5: 課金方法の設定

### 操作手順

1. X Ads Manager → **「アカウント設定」 → 「課金」**
2. **「+ 支払い方法を追加」**
3. 推奨: **法人クレジットカード**（請求書方式は月額消費額が一定以上必要）
4. 通貨: **JPY**
5. デフォルト課金方式: **クレジットカード**

> **初心者ポイント**: X Ads は **前払い型**ではなく後払い型（月次請求）。請求書ベースに移行するには通常3-6ヶ月の利用実績が必要。最初はクレジットカードで。

## トラブルシュート

| 症状 | 原因候補 | 対処 |
|---|---|---|
| 申請が「審査中」のまま48h以上 | アカウント信頼度不足・国/業種の整合性 | サポート問い合わせ・別アカウントで再申請検討 |
| 「ポリシー違反」で申請却下 | 投稿実績ゼロ・名義不一致 | アカウントで投稿してから再申請 |
| Pixel イベントが計上されない | タグ未実装 or イベント発火条件のミス | PR で実装確認・テンイチ画面でテスト |
| 課金方法が登録できない | 法人クレジットカード未承認 | サポート問い合わせ |
