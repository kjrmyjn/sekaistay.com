# X Ads コピー下書き

> **対象**: X（旧 Twitter）プロモーション広告
>
> **配信目的**: ウェブサイトへのコンバージョン（Lead）
>
> **媒体特性**: テキスト主体（280文字）、若年〜中年男性ヘビーユーザー、リアルタイム性。**民泊系は他媒体より CPA 高い傾向**（¥10,000-20,000 想定）。

---

## クリエイティブ構成要素（X フォーマット）

| 要素 | 文字数制限 | 役割 |
|---|---|---|
| **ツイート本文** | 280 文字 | 本文・URL は1リンク扱い（23文字消費） |
| **画像** | 1200×675 推奨（16:9） | スクロール停止訴求 |
| **動画** | 15-60秒、最大 1分 | 高エンゲージメント |
| **Website Card（リンク先プレビュー）** | OG タグ自動取得 | ヘッドラインとボタン文言 |
| **CTA ボタン** | 既定文言（「詳細はこちら」「申し込む」等） | クリック誘導 |

> **初心者ポイント**: X は **画像・動画なしで広告配信不可**。最低1点の画像が必須。

## Website Card 設定

X 広告では **Website Card** という表示形式があり、ツイートの下に画像 + ヘッドライン + CTA ボタンが表示される。

| 項目 | 推奨設定 |
|---|---|
| **画像** | 1200×675・既存ブランド画像（feature-support.jpg 等） |
| **ヘッドライン** | 「成果で選ばれる民泊運用代行」 |
| **CTA ボタン** | 「詳細はこちら」 |

ヘッドラインのバリエーション案:
- 「成果で選ばれる民泊運用代行」
- 「手数料8%・業界相場の半額」
- 「レビュー4.8の運用代行」
- 「30秒で無料診断」

## ハッシュタグ戦略

> **方針**: ハッシュタグは **広告ツイートには最大2個**。多すぎると広告感が強まりエンゲージ低下。

| ハッシュタグ | 検索ボリューム想定 | 用途 |
|---|---|---|
| `#民泊運用代行` | 中 | カテゴリ明示 |
| `#Airbnb` | 大 | 広い認知 |
| `#民泊` | 大 | 同上 |
| `#不動産投資` | 大 | 投資家層リーチ |
| `#民泊オーナー` | 小 | ニッチ精度 |

推奨組み合わせ:
- カテゴリ訴求: `#民泊運用代行 #Airbnb`
- 投資家向け: `#民泊 #不動産投資`
- リターゲ: なし（ハッシュタグなし）

## 画像素材の方針

### Meta と同じ既存素材を流用

`IMAGES_MANIFEST.md` から:
- `feature-support.jpg` — ワンストップ訴求用
- `feature-inbound.jpg` — グローバル/インバウンド訴求
- `svc-pricing.jpg` — ダイナミックプライシング訴求

### X 特有の画像最適化

- **テキスト挿入**: 画像内に大きな数字（「8%」「4.8」「+18%」）を入れると視認性 ⤴
- **画面切り取り風**: ダッシュボードのスクリーンショット風（情報量訴求）
- **比較画像**: 「業界相場 15-20% / SEKAI STAY 8%」を並列表示

## A/B テスト設計

> X はアルゴリズム学習が Meta より弱いので、**手動でクリエイティブを切り替えながら勝者を見つける** 方が早い。

| 週 | 試すパターン |
|---|---|
| 第1週 | A-1, B-1, D-1（短長×訴求軸の組み合わせ） |
| 第2週 | 第1週の勝者×新コピー2本 |
| 第3週以降 | 安定したクリエイティブを Meta の勝ったコピーで上書きテスト |

## ランディング URL

```
https://sekaistay.com/switch?utm_source=x&utm_medium=cpc&utm_campaign={campaign}&utm_content={creative_name}        ← Control
https://sekaistay.com/switch/short?utm_source=x&utm_medium=cpc&utm_campaign={campaign}&utm_content={creative_name}   ← D (45min meeting)
```

> **X 推奨**: 月¥150K 予算で 15 リードを狙うため、variant を絞る方が良い。**Control + D（45分面談）の 2 variants** からスタート。X はテキスト主体で短文と CTA が効きやすいので /switch/short と相性が良い。

## X 特有の注意

| 項目 | 注意点 |
|---|---|
| **広告ポリシー** | 「業界相場の半額」のような比較表現に厳しい。差別比較を避けて「業界相場と比べて〜」と表現を緩める方が承認されやすい |
| **薬機法・特商法系の文言** | 民泊系では問題ないが、リード起票後の運用で「保証収入」「絶対」等は使わない |
| **画像内テキスト比率** | 古い 20% ルールはなくなったが、テキスト多すぎるとクリック率低下 |
| **再投稿** | 同じツイートを使い続けるとクリエイティブ疲弊が早い → 週1で新クリエイティブ投入推奨 |
